SimoReconnect は、Velocity プロキシサーバー専用の軽量な再接続管理プラグインです。プレイヤーがバックエンドサーバーから切断された際、タイトル画面に戻されるのを防ぎ、自動的に指定したフォールバックサーバー（ロビーなど）へ転送します。

🌟 主な機能
自動フォールバック: 接続中のサーバーがダウンしたり再起動したりした際、プレイヤーをキックせずに別のサーバー（Lobby等）へ即座に移動させます。

再接続ロジックの最適化: Velocity 標準の挙動をカスタマイズし、より安定した再接続体験を提供します。

カスタマイズ可能なメッセージ: サーバー移動時や再接続時に、プレイヤーに対して状況を説明するメッセージを表示可能です。

✨ 導入のメリット
プレイヤーの離脱防止: サーバーの不意な切断による「タイトル画面戻り」を無くすことで、プレイヤーがサーバーに留まりやすくなります。

メンテナンスの円滑化: バックエンドサーバーをメンテナンスモードにする際、プレイヤーを安全にロビーへ退避させることができます。

🔧 導入方法
ビルドされた .jar ファイルを Velocity サーバーの plugins フォルダに配置します。

Velocity を起動します。

生成された config.toml（または .json）で、再接続先の優先順位やメッセージを設定します。

📋 動作要件
Proxy: Velocity 3.x 以上

Type: Proxy-side Only
